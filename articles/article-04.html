Canvas Aim Trainer


This browser app acts as an aim trainer.


This project utilizes canvas to implement an aim trainer. Canvas with HTML and JavaScript makes it easy to implement graphical applications on your browser. In this case, I can easily create a dynamic game that utilizes different views that change with the defined controls (mouse events such as clicking or pressing the escape key in this application). The aim trainer is separated into two different modes: menu and game. The trainer starts in menu mode where it tells the user to click in the middle of the screen. Once the user clicks the middle of the screen, the trainer will switch to game mode where targets remaining and number of missclicks will be displayed and three targets will always be displayed. The user will click on these targets until user hits thirty targets. Once the user hits thirty targets, the number of targets hit, accuracy, missclicks, and average time between clicks will be displayed. Accuracy is determined by the number of targets hit (30) and number of missclicks. The average time between clicks is determined by recording the animation frame time of a click, taking the different times between each click, taking the sum of the differences, and dividing by how many of these differences there are. Targets are drawn using canvas, and a helper random function that will determine where the targets are drawn randomly. The canvas is viewed and updated in the browser using the requestAnimationFrame method, and this is key in finding the time between clicks. Pressing the escape key will send the user back to menu view.
