Canvas Aim Trainer with Leaderboard


This browser app mainly uses the same logic as my aim trainer app, but the logic in this one has a new mode called end. This mode removes the event listeners for controls so that the user can not change their score in any way. Also, the accuracy and time between clicks variables are instantiated under state. This makes it so that these variables can be used to implement the leaderboard. For the leaderboard, lower time between clicks = better since it means that one score is faster than the other. The leaderboard displays the top 5 times using the module design pattern.
